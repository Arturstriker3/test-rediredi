# Repositório para teste técnico RediRedi.

## Clonar Repositório
```
git clone https://github.com/Arturstriker3/test-rediredi
```

## Abrir Repositório
```
cd test-rediredi
```

## Instalar Dependências
```
pnpm install
```

## Rodar localmente:
```
pnpm run dev
```
